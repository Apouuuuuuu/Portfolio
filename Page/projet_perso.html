Projet 1 : Création d'un bot de modération discord de modération.


Ce projet est la réalisation d'un bot Discord.

Les bots discord peuvent être assez complets. J'ai décidé de dédier celui-ci à la modération.

De nombreuses fonctions sont disponnible sur ce bot, comme l'automatisation de certaines commandes (bannissement et exclusion d'utilisateur, commande de giveway, intelligences artificielles (avec l'utilisation de l'API de Open AI)...)

Ce bot est actuellement hébergé par un service d'hébergement tiers.



Langages utilisés

Javascript
95%

SQL
5%

Choix des langages

Pour ce bot discord j'ai essentiellement utilisé du Javascript étant donné que la documentation de NodeJS et DiscordJS est très large.
Le python peut également être utilisé pour la création de bot discord mais je me sens moins à l'aise avec ce langage.
De plus, j'ai utilisé du SQL pour effectuer mes requêtes à ma base de donnée qui est nécessaire pour certaines fonctions, tels que le système d'expérience et le système d'avertissement.

Projet 2 : Création d'un site web


Ce site web est mon projet de fin d'année de BTS. Je devais créer une application web (avec les outils que je voulais) de covoiturage.

Ce projet à été réalisé durant mon temps libre à l'aide des différentes connaissances acquises durant ma formation.
Mon but dans ce projet était de faciliter la gestion du co-voiturage. L'objectif était de faciliter la tâche des employés avec un site web compréhensible, facile d'accès et fonctionnel.
Il y à plusieurs fonctionnalités sur le site, comme le fait de rechercher un trajet, gérer ses réservations, donner un avis sur une réservation, modifier son profil, l'exportation de toutes les données...



Langages utilisés

PHP
73.5%

SQL
10%

HTML
10%

CSS
6.5%

Choix des langages

Pour ce site web j'ai utilisé plusieurs langages.
Le PHP est celui qui prend la plus grosse partie. J'ai décidé d'utiliser le php pour le back-end puisque c'était un langage assez facile d'accès.
J'ai également utilisé du HTML et du CSS pour la partie front-end. J'ai d'ailleurs utilisé un framework pour le CSS : Bootstrap Enfin, j'ai utilisé du SQL pour mes requêtes de base de données qui étaient nécessaires pour la quasi totalité des fonctions (connexion, inscription, avis, poste d'un trajet...)

Projet 3 : Création d'un bot d'alerte de stream Twitch.


Ce projet est la réalisation d'un bot discord spécialisé dans les alertes de stream sur Twitch

L Étant donné que j'avais déjà des bases pour les bot discord, je me suis lancé dans un bot discord avec l'API de Twitch. Le bot à pour but d'alerter des utilisateurs quand un streamer (qui à été ajouté au bot) lance un live sur la plateforme Twitch.

Ce bot est actuellement hébergé par un service d'hébergement tiers.



Langages utilisés

Javascript
100%

Choix des langages

Pour ce bot discord j'ai uniquement utilisé du Javascript étant donné que la documentation de NodeJS et DiscordJS est très large et que j'avais déjà des connaissances sur ce langage, particulièrement pour DiscordJS.
Je n'ai pas eu besoin d'utiliser de SQL pour ce bot, puisque tous les streamers qui sont enregistrés sur la commande sont ajoutés dans un fichier .json, je n'avais donc pas d'utilité à ajouter de base de donnée.
