Projet : Création d'un site web


Ce projet à été réalisé durant mon apprentissage.
Mon but dans ce projet était de faciliter l'organisation des différents outils informatique de mon entreprise, la Mutualité française de Haute Saône.

3 grands points étaient imposés :
- l'organisation de sortie de parc des ordinateurs
- l'organisation de l'adressage IP des outils informatiques
- l'organisation des imprimantes du parc informatique

De base, tout était géré sur de grands excels peu pratiques et assez compliqués à comprendre.

J'ai donc réalisé un site qui englobe ces 3 fonctions.
J'ai évidemment ajouté d'autres fonctionnalités tel que des filtres par date, la modification de toutes les données, la sécurisation du site, l'exportation de toutes les données, la connexion ELDAP, l'hébergement du site...

Choix des langages

Pour ce site web j'ai utilisé plusieurs langages.
Le PHP est celui qui prend la plus grosse partie. J'ai décidé d'utiliser le php pour le back-end puisque c'était un langage assez facile d'accès.
J'ai également utilisé du HTML et du CSS pour la partie front-end. J'ai d'ailleurs utilisé un framework pour le CSS : Bootstrap Enfin, j'ai utilisé du SQL pour mes requêtes de base de données qui étaient nécessaires pour la quasi totalité des fonctions (connexion, inscription, avis, poste d'un trajet...)
